[0.7.0]

  * Added several object generators to the 'sketchy' pack: carpets, bathroom items...
  * Trying to normalize slots (DDDSlot) and added very basic book generation.
  * Added centerline() method to DDDNode2.
  * Fixes to support working on up to date Docker images.
  * Removed usage of geom.type (deprecated by Shapely in favour of geom.geom_type).
  * Fixed rotation including support for transforms and instances and path3. 
  * Added axis transform support for export.
  * Changed vertex_func() to support world coordinates (breaking change: causes regressions).
  * Fixes to geometry split and other operations. Added missing features and cases to existing operations.
  * Large changes to unify DDDObject 2 and 3 into DDDNode.
  * Separated DDDMaterial from ddd.
  * Fixes to path height perp interpolation and object dump repr.
  * Added path height, and some severe changes to group2/3 (now allowing mixing types).
  * Adding support for path entities (arcs, bezier) and new path-height function which correctly interpolates height on curves.
  * Separated logo item to pack.
  * Added FBX export (through Blender).

[0.6.5]

  * Updated Shapely version.
  * Fixes to SVG export and added SVG text export.
  * Added vendorized/custom bakefont3 lib.
  * Added lamp_mount=high_mast.
  * Large changes to interways building, and ways and areas containment processing.
  * Several fixes to building contacts calculations, and workarounded level 0 footprint error.
  * Adding support for 2D (font atlas based) text.
  * Adapt osm_tile_info as pipeline.
  * Added support for metallic-roughness maps, and missing socketio dependency.
  * Adding support for no DEM, changing settings/props.
  * Several changes to params/settings/paths to adapt to osm_build as pipeline.
  * Added time and object count info to pipeline tasks.
  * Introducing dddserver.
  * Added pycatenary and cable catenaries for power lines.
  * Subdivide to grid now handles UV correctly.
  * Fixed missing splatmap tag for combined materials.
  * Added population database source, and augmentation for building levels using it.
  * Added lava, Canary Islands, emissive textures.
  * Added draco compression support. Added plane split method.

  * Fixes to building fully-filled fix. Severe bug fix in centroid().
  * Fixed GeoJSON metadata export so DDD extra attributes are present in the feature metadata.
  * Flattening before combine_group().
  * Fixed extrude_coords support for lines and points, and now reusing closing vertices.
  * Fixes invalid triangles generated by extrude_coords (wrap extrusion method).
  * Added support for object combining via group criteria. Marking splat materials as such.
